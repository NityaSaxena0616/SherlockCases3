# Sherlock Holmes: Interactive Cases ‚Äì Enhanced Edition

An immersive 3D web experience inspired by the legendary detective Sherlock Holmes.  
Built using **Three.js** and **GSAP**, this project recreates iconic Holmes stories with cinematic visuals, character interactions, and an interactive mystery-solving system.

---

##  Features

-  3D atmospheric environment (fog, dust, volumetric lighting)
-  Animated 3D characters (Sherlock, Helen, Roylott)
-  Cinematic subtitles & dialogue sequencing
-  Audio toggle support
-  Interactive suspect selection system
-  Mystery reveal & explanation engine
-  Smooth camera transitions & mouse parallax effect
-  Dynamic lighting & candle flicker animation

---

##  Included Cases

- The Speckled Band (1892)
- A Study in Scarlet (1887)
- The Hound of the Baskervilles (1902)
- A Scandal in Bohemia (1891)

Each case includes:
- Guided dialogue storytelling
- Character interactions
- Suspect selection
- Correct/incorrect deduction reveal

---

##  Technologies Used

- HTML5
- CSS3 (Advanced animations, cinematic styling)
- JavaScript (ES6)
- Three.js (3D rendering)
- GSAP (Animation library)

CDN Dependencies:
- Three.js r128
- GSAP 3.12.2

---

##  How to Run

1. Download or clone the project.
2. Open the HTML file in a modern browser (Chrome recommended).
3. For best performance, use a local server:

   Example using VS Code Live Server:
   - Right-click HTML file
   - Click "Open with Live Server"

---

##  How to Play

1. Wait for the cinematic loading screen.
2. Select a Sherlock Holmes case.
3. Watch the story unfold through dialogue.
4. Click characters to focus on them.
5. Choose who you believe is the culprit.
6. Reveal the answer and explanation.
7. Restart to try another case.

---

##  Performance Notes

- Optimized fog particle systems
